Intranet, Extranet e Internet



Internet

  • Rede mundial que conecta computadores e dispositivos.
  • Permite acessar sites, redes sociais e e-mails.

Exemplo: Quando você acessa o YouTube para assistir a um vídeo.


Intranet

  • Rede privada de uma empresa ou organização.
  • Somente funcionários podem acessar.

Exemplo: Um site interno de uma escola onde professores postam notas dos alunos.


Extranet

  • Uma intranet com acesso restrito para usuários externos.
  • Permite colaboração entre empresas.

Exemplo: Uma loja que compartilha informações de estoque com fornecedores.


World Wide Web


A Web (World Wide Web) é um sistema de informações interligadas acessíveis pela Internet. Imagine a Web como uma grande biblioteca digital onde você pode acessar sites, assistir vídeos, fazer compras e interagir com outras pessoas.

Exemplo: Quando você pesquisa no Google, clica em um link e abre um site, está navegando na Web!

Sites e aplicações web


Sites

Um conjunto de páginas acessadas por um navegador.

Exemplo: Um blog com artigos sobre tecnologia.


Paginas Web

Arquivo estático contendo textos, imagens e outros elementos.


Aplicações Web

Software interativo acessado via navegador, permitindo operações como cadastro e login.

Exemplo: O Google Drive, onde você pode criar e editar documentos online.


Navegadores e Dispositivos


Navegadores

Programa que interpreta e exibe páginas da Web (ex.: Chrome, Firefox, Edge).


Dispositivos

Computadores, smartphones, smart TVs e outros meios de acesso à internet.


Arquitetura Web


Cliente-servidor

  • Cliente: O navegador solicita informações.
  • Servidor: Responde enviando conteúdos, como páginas e imagens.

Uniform Resource Locator (URL)

O URL é o endereço de um recurso na Web, composto por: Protocolo, domínio e caminho

  • https://

    → Protocolo de segurança.

  • www.google.com

    → Nome do site.

  • /search?q=wikipedia

    → Página de pesquisa.

Protocolos

  • HTTP/HTTPS: Transferência de páginas da Web.
  • FTP: Transferência de arquivos.
  • SMTP: Envio de e-mails.
  • TCP/UDP: Controle de transmissão de dados.
